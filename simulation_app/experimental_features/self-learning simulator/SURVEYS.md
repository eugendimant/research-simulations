# Survey primitives (v0.16)

Implemented survey methods (response-level simulation, action-only):
- Likert (ordered latent + noise)
- Binary conjoint choice (utility + noise)
- List experiment (count outcome)
- Randomized response (privacy-preserving binary)
- Endorsement experiment (endorsement shift + latent attitude)

Run surveys through `socsim.surveys.runner.simulate_survey`.
